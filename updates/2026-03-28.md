# 2026-03-28 Changes

## Security Fixes

### 1. `ADMIN_API_KEY` empty-value bypass bug fix
**File:** `backend/src/handlers/deleteCat.ts`

```diff
- if (ADMIN_KEY && event.headers['x-admin-key'] !== ADMIN_KEY) {
+ if (!ADMIN_KEY || event.headers['x-admin-key'] !== ADMIN_KEY) {
```

Returns 403 unconditionally when `ADMIN_KEY` is not set.

---

### 2. Remove CORS wildcard + add `x-admin-key` to allowed headers
**File:** `backend/src/lib/response.ts`

- Throws an error at Lambda init time when `ALLOWED_ORIGIN` is not set (removes `*` fallback)
- Added `x-admin-key` to `Access-Control-Allow-Headers`

---

### 3. Add security headers
**File:** `backend/src/lib/response.ts`

Added to all responses (`ok`, `err`, `options`):
- `X-Content-Type-Options: nosniff`
- `X-Frame-Options: DENY`
- `Strict-Transport-Security: max-age=31536000; includeSubDomains`

---

### 4. Add cursor parameter schema validation
**File:** `backend/src/handlers/getCats.ts`

- Added `parseCursor()` function
- Allowed keys: `PK`, `SK`, `town`, `uploadedAt` only
- Allowed value types: string or number only
- Returns 400 on parse failure or unexpected keys/types

---

### 5. Add API Gateway CloudWatch access logging
**File:** `infrastructure/modules/api/main.tf`

- Created CloudWatch log group `/aws/apigateway/{project}-{environment}` (30-day retention)
- Added `access_log_settings` to the `v1` stage
- Logged fields: requestId, requestTime, httpMethod, routeKey, status, responseLength, sourceIp, userAgent, errorMessage
- Requires `terraform apply` to take effect

---
